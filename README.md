# BikeDekho Project

## Description
BikeDekho is a web-based application developed using PHP and MySQL
that provides bike details, management features, and user interaction.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Features
- Bike listings
- Bike details view
- Admin management
- Database connectivity

## How to Run the Project
1. Install XAMPP
2. Copy the project to `htdocs`
3. Start Apache and MySQL
